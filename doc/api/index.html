Playify

Playify is a Flutter plugin for playing music and fetching music metadata. Supports only iOS. Still in development. A detailed README will be added in the future. Checkout the Development Log.

Requirements:

  • iOS 10.1 (Due to MPMusicPlayerMediaItemQueueDescriptor)
